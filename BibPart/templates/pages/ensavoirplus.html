Cette application a été développée par Fanny Lebreton dans le cadre du Master 2 Technologies numériques appliquées à l'histoire de l'École nationale des chartes.


Intérêts scientifiques de BibPart


BibPart a pour objectif de recenser des partitions manuscrites authographes afin de centraliser ce type de source pour en faciliter l'accès et la recherche.

Cette application s'adresse aux chercheurs, car les partitions originales sont de véritables références pour la musicologie. Elles leur permettent par exemple d'analyser le processus de composition d'un compositeur. Elles peuvent être aussi une source intéressante d'un point de vue matériel (intérêt pour l'encre, le papier et l'écriture manuscrite).

BibPart peut être utile aussi pour les éditeurs de musique. Ces derniers pourraient consulter cette application afin de se référer à ces sources originales lorsqu'ils éditent une partition.

Enfin, BibPart s'adresse à tous musiciens et mélomanes curieux, désireux de découvrir l'écriture d'un compositeur.



Fonctionnement de BibPart


Comment est construit BibPart ?

L'application a été créée avec les langages Python 3 (framework Flask), HTML et CSS (framework Bootstrap), ainsi qu'à partir d'une base de données SQLite.


Quelles sont ses fonctionnalités ?

3 index rencensent l'ensemble des données enregistrées dans la base :

  • l'index des oeuvres
  • l'index des partitions
  • l'index des compositeurs

Chaque enregistrement repertorié dans l'index renvoie vers une notice présentant des données bibliographiques.

Une recherche rapide est disponible et permet de donner accès à la notice bibliographique d'une oeuvre. Elle peut être trouvée à l'aide de mots clefs portant sur le titre, la date, le type, la forme de l'oeuvre, et sur le compositeur.

L'utilisateur a la possibilité de s'inscrire, puis de se connecter afin de pouvoir avoir accès à d'autres fonctionnalités.

En effet, une fois connecté, l'utilisateur peut contribuer à faire évoluer le contenu de la base de données :

  • il peut enrichir la base grâce au formulaire Enrichir la base.
  • il peut modifier les données grâce au formulaire Modifier accessible sur chacune des notices.
  • il peut supprimer les données grâce au formulaire Supprimer accessible également sur chacune des notices.

Il peut également effectuer une recherche avancée, lui permettant de trouver à partir de différentes entrées, soit la notice de l'oeuvre, soit la notice de la partition, soit la notice du compositeur.

Installation de l'application


Les informations liées à l'installation de BibPart sont disponibles sur Github. Il est recommandé de créer d'abord un environnement virtuel, puis d'installer les dépendances du projet via le fichier requirements.txt. Il est ensuite possible de lancer l'application grâce au fichier run.py.

Il est recommandé d'être connecté à une connexion internet afin d'avoir accès aux visionneuses des partitions et aux vidéos.
