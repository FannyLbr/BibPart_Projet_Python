Aide à l'ajout

Ce présent formulaire permet d'enrichir la base existante avec l'enregistrement de nouvelles données.

Trois démarches sont possibles :

    Cas 1 : vous souhaitez ajouter une oeuvre et sa partition dont aucune donnée n'est présente dans la base : nous vous conseillons de remplir l'ensemble des formulaires dans l'ordre.
    Cas 2 : vous souhaitez ajouter une oeuvre et sa partition dont certaines données sont déjà présentes dans la base : nous vous conseillons de ne remplir que les formulaires permettant d'ajouter les nouvelles données.
    Cas 3 : vous souhaitez ajouter une partition se rattachant à une oeuvre déjà présente dans la base : nous vous conseillons de ne remplir que le formulaire de la partition.

Dans l'ensemble des formulaires, certains champs sont marqués d'un *, cela signifie qu'ils sont obligatoires.

Les formulaires pour l'ajout d'une oeuvre et l'ajout d'une partition ont des champs avec des listes, il suffit de sélectionner la donnée souhaitée.
